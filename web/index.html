PrintOrb

Flash the firmware to your orb — right from this browser.

⚠️
Your browser can’t talk to the device over USB. Web flashing needs the Web Serial API — please use Chrome, Edge, or Opera on desktop (not Firefox, Safari, or a phone).
Web Serial isn’t available — see the note above. You need to allow the serial port to continue.
Plug the board in via USB-C, click the button, pick the USB-Serial CH343 port, and let it flash. ~30 seconds.

How it works

  1. Connect the board to your computer with a USB-C data cable.
  2. Click Connect & Install and choose the serial port that appears.
  3. Confirm the install — the orb is fully erased and flashed with the latest firmware.
  4. On first boot it opens a Wi-Fi setup hotspot printorb-setup-xxxx; join it to configure your printer.
Stuck in the bootloader? Hold BOOT, tap RESET, release BOOT, then retry. Already running PrintOrb? You can also update wirelessly at printorb.local → Update tab.
